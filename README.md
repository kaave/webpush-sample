# Webpush sample

## How to use

### yarn (recommend)

```bash
# install packages
yarn
# install sw & push subscription
yarn start
# add .env file
cp .env.sample .env
# edit .env
vi .env
# send push notification
yarn push
```

### npm

```bash
# install packages
npm i
# install sw
npm start
# add .env file
cp .env.sample .env
# edit .env
vi .env
# send push notification
npm run push
```
